# Getting Started with Go (Golang)

Welcome back to our Go (Golang) exploration journey! In this second part of our series, we'll dive deeper into the language. By the end of this post, you'll have a solid foundation in Go's basic syntax, data types, and control structures.

## Basic Syntax

Let's start by taking a look at some fundamental aspects of Go's syntax.

### **Hello World in Go**

In Go, creating a "Hello World" program is delightfully simple:

```go
package main

import "fmt"

func main() {
    fmt.Println("Hello, World!")
}
